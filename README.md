# vue2.0仿同城生活通

__注：该项目还是进一步完善，其中有很多功能没有走通，后续会持续更新补全

> 基于Vue2.0改版的O2O同城生活通中型单页应用项目



## 项目运行

``` bash
git clone https://github.com/qiheng/vue2-o2o.git

cd vue2-o2o

# 依赖
npm install

# 编译环境 localhost:8089
npm run dev

# 线上版本
npm run build

```
