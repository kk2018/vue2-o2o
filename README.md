## 项目运行

``` bash
git clone https://github.com/qiheng/vue2-o2o.git

cd vue2-o2o

# 依赖
npm install

# 编译环境
npm run dev

# 线上版本
npm run build

```

访问 ![http://localhost:8089](http://localhost:8089)

## 技术栈
vue2 + vuex + vue-router + webpack + ES6/7 + axios + less + vux
